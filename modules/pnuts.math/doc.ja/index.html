pnuts.math モジュール

pnuts.math モジュールには、java.lang.Math クラスに定義されている算術演算に対応する関数が含まれています。詳しくは、APIドキュメントを見てください。

sin ( double d )

指定された角度の正弦 (サイン) を返します。

cos ( double d )

指定された角度の余弦 (コサイン) を返します。

tan ( double d )

指定された角度の正接 (タンジェント) を返します。

asin ( double d )

指定された角度の逆正弦 (アークサイン) を、-pi/2 〜 pi/2 の範囲で返します。

acos ( double d )

指定された角度の逆余弦 (アークコサイン) を、0.0 〜 pi の範囲で返します。

atan ( double d )

指定された角度の逆正接 (アークタンジェント) を、-pi/2 〜 pi/2 の範囲で返します。

atan2 ( double d1 , double d2 )

直交座標 (x, y) を極座標 (r, theta) に変換します。

toRadians ( double d )

度で計測した角度を、相当するラジアンに変換します。

toDegrees ( double d )

ラジアンで計測した角度を、相当する度に変換します。

exp ( double d )

オイラー数 e を double 値で累乗した値を返します。

log ( double d )

指定された double 値の自然対数値 (底は e) を返します。

sqrt ( double d )

double 値の正しく丸めた正の平方根を返します。

IEEEremainder ( double d1, double d2 )

IEEE 754 標準に従って、2 個の引数について剰余を計算します。

ceil ( double d )
floor ( double d )

引数の値以下で、計算上の整数と等しい、最大の (無限大にもっとも近い) double 値を返します。

rint ( double d )

引数の値にもっとも近く、計算上の整数に等しい double 値を返します。

pow ( double d1 , double d2)

1 番目の引数を、2 番目の引数で累乗した値を返します。

round ( double d )

引数にもっとも近い int を返します。

abs ( Number d )

絶対値を返します。

max ( d1, d2 ) or
( collectionOrArray {, comparator | f(arg) })

2 つの値のうち大きい方を返します。

1番目の引数にCollectionか配列が指定された場合は、その中の最大の要素を求めます。 comparatorやfは各要素の比較のために使われます。 

max([1,2,-6,4], abs)  -> -6
min ( d1, d2 )or
( collectionOrArray {, comparator | f(arg) })

2 つの値のうち小さい方を返します。

1番目の引数にCollectionか配列が指定された場合は、その中の最小の要素を求めます。comparatorやfは各要素の比較のために使われます。 

min([1,2,-6,4], abs)  -> 1
