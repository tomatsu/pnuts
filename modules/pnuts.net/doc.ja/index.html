getLocalHost()

getLocalHost() は、ローカル・ホストのアドレスを表すInetAddressオブジェクトを返します。

getInetAddress(String host, String address) or
(String hostOrAddress)

getInetAddress() は、指定されたホストのアドレスを表す InetAddressオブジェクトを返します。

例： 
getInetAddress("127.0.0.1")
getInetAddress("pnuts.org")
isLocalHost(InetAddress address) or
(String hostOrddress)

isLocalHost() は、指定されたアドレスがローカルホストのアドレスに含まれるかどうかをp調べます。

例： 
isLocalHost("127.0.0.1")
isLocalHost("pnuts.org")
encodeURL ( String str {, String encoding } )

encodeURL()は、生テキストからx-www-form-urlencodedフォーマットのテキストに変換します。

encodingには、Javaのエンコード名を指定します。 encodingを指定しない場合は、デフォルトエンコーディングが使われます。

例: 
encodeURL("How are you?", "UTF-8") ==> "How+are+you%3F"
decodeURL ( String encoded {, String encoding } )

decodeURL()は、x-www-form-urlencodedフォーマットのテキストから生テキストに変換します。

encodingには、Javaのエンコード名を指定します。 encodingを指定しない場合は、デフォルトエンコーディングが使われます。

例: 
decodeURL("How+are+you%3F", "UTF-8") ==> "How are you?"
makeQueryString ( Map map , String encoding )

makeQueryString()は、{パラメータ, 値}のペアを持つMapオブジェクトから、x-www-form-urlencodedフォーマットのQuery Stringを作ります。
