Configuration に関するサポート

Public以外のメンバーへのアクセス

nonPublicMemberAccess( { boolean enable })

trueが渡されると、それ以降、publicでないメンバーにアクセスできるようになります。 逆に、falseは、publicでないメンバーへのアクセスを無効にします。

パラメータがない場合、この関数は、publicでないメンバーにアクセスできるかどうかを返します。

例： 
nonPublicMemberAccess() ==> false
nonPublicMemberAccess(true)
"hello".value ==> ['h', 'e', 'l', 'l', 'o']
nonPublicMemberAccess() ==> true

Publicメンバーへのアクセス

publicMemberAccess( { boolean enable })

trueが渡されると、それ以降、publicメンバーにアクセスできるようになります。 逆に、falseは、publicメンバーへのアクセスを無効にします。

パラメータがない場合、この関数は、publicメンバーにアクセスできるかどうかを返します。

例： 
publicMemberAccess(true)
import java.awt.Point
pt = Point(1,2)
println(pt.x, ", ", pt.y)
