Pnuts ユーザーズガイド -> pnuts.util -> Manifest情報の取得

Manifest情報の取得

manifest( String module_name { , handler( key, value ) } ) or
(Class class { , handler( key, value ) } ) or
(File file { , handler( key, value ) } ) or
(JarFile jarfile { , handler( key, value ) } )

manifest()は、クラスやモジュールが含まれるJARファイルからManifest情報を取得して、その内容を表示します。

handlerに2引数の関数が指定された場合は、Manifestの属性とその値を引数にしてその関数が呼び出されます。

例(1): 
manifest("pnuts.lib")

例(2): 
m = map()
manifest(class pnuts.lang.Pnuts, function (k, v) m.put(k, v))

モジュールのバージョン情報

versionInfo(String module_name) or
(Class class)

versionInfo()は、指定されたクラスやモジュールのバージョン情報を表すjava.lang.Packageオブジェクトを返します。モジュール名を指定する場合は、初期化用のクラスが必要になります(つまり、init.pnutでなくinit.classが必要)。

この関数はJDK1.2以降が必要です。

例(1): 
versionInfo("pnuts.lib")
例(2): 
versionInfo(class pnuts.lang.Pnuts)
