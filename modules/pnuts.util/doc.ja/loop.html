ループ

loopEnum(int n)

loopEnum()は、n回ループするjava.util.Enumerationオブジェクトを返します。

例: 
for (always_null : loopEnum(1000)){
    ..
}
rangeEnum(int start , int end {, int step } )

rangeEnum()は、startからendまでの整数を要素 として返すjava.util.Enumerationオブジェクトを返します。 stepが指定された場合は、返される要素は、start, start + step, start + step * 2, ... のようになります。 stepが指定されない場合のstep値は、startがendよりお おきい場合は-1、それ以外は1になります。

例: 
for (i : rangeEnum(0, 100, 30)){
   println(i)
}

0
30
60
90
