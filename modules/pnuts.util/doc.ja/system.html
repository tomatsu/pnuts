プラットフォームのコマンド呼出し

system(String platformCommand {, (Reader | InputStream) input {, (Writer|OutputStream) output } } )

system()は外部コマンドを実行して結果を表示します。第2引数inputが指定された場合は、そのストリームから読み込まれたデータが標準入力に書き込まれます。

例1: 
> system("ls -l")
...
例2 
> system("wc", open("file.txt"))
