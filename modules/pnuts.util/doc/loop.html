Loop

loopEnum(int n)

loopEnum() returns a java.util.Enumeration object that can be used for n loop used for a n loop.

e.g: 
for (always_null : loopEnum(1000)){
    ..
}
rangeEnum(int start , int end {, int step } )

rangeEnum() returns a java.util.Enumeration object that returns integers from start to end as the elements. When step is specified, the returned elements would be start, start + step, start + step * 2, ... . When step is not specified, the implicit value of step is -1 when start is larger than end, otherwise 1.

e.g: 
for (i : rangeEnum(0, 100, 30)){
   println(i)
}
