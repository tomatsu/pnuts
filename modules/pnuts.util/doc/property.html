Properties

System Properties

getProperty(String property)
setProperty(String property , Object value)

getProperty() gets the value of a system property. setProperty() defines a system property.

e.g. 
> getProperty("java.version")
1.1.5

> getProperty("line.separator").toCharArray()
['\n']
isJava2()

isJava2() returns true if the running JVM is compliant with J2SE, J2EE, or J2ME/CDC.

Reading Properties

loadProperties(String resource {, Properties properties })
(File file {, Properties properties })
(InputStream input {, Properties properties })
(URL url {, Properties properties })

loadProperties() reads properties from a resource file, a local file, an inputStream, or a URL. If properties is specified the loaded properties are merged into the properties.
