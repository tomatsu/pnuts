Calling Platform Commands

system(String platformCommand {, (Reader | InputStream) input {, (Writer|OutputStream) output } } )

system() executes an external command and prints the result. If the 2nd parameter input is specified, the data read from the stream is written to the standard input of the command.

e.g. (1) 
> system("ls -l")
...
e.g.(2) 
> system("wc", open("file.txt"))
