Configuration Support

Accessing Non Public Members

nonPublicMemberAccess( { boolean enable } )

If true is given as the parameter, subsequently non-public members can be accessed. Contrary, false disables the non-public member access.

If no parameter is specified, this function returns if non-public member access is enabled or not.

e.g: 
nonPublicMemberAccess() ==> false
nonPublicMemberAccess(true)
"hello".value ==> ['h', 'e', 'l', 'l', 'o']
nonPublicMemberAccess() ==> true

Accessing Public Members

publicMemberAccess( { boolean enable })

If true is given as the parameter, subsequently public members can be accessed. Contrary, false disables the public member access.

If no parameter is specified, this function returns if public member access is enabled or not.

e.g. 
publicMemberAccess(true)
import java.awt.Point
pt = Point(1,2)
println(pt.x, ", ", pt.y)
