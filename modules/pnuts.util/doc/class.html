Inspecting Class Definition

dumpclass(Class class)

dumpclass() displays all public members of class.

e.g: 
> dumpclass(Object)

--- constructors  -----------------------
 Object()

--- instance methods -----------------------
 Class getClass()
 int hashCode()
 boolean equals(Object)
 String toString()
 void notify()
 void notifyAll()
 void wait(long)
 void wait(long,int)
 void wait()
supertypes(Class class)

supertypes() displays all superclasses and interfaces of class.

e.g: 
> supertypes(class java.util.Vector)

java.util.AbstractList class
java.util.AbstractCollection class
java.lang.Object class
java.util.List interface
java.util.Collection interface
java.lang.Cloneable interface
java.io.Serializable interface
java.util.List interface
java.util.Collection interface
java.util.Collection interface
null
