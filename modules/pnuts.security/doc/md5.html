Message Digest - MD5, SHA

md5(byte[] array {, int offset , int length } ) or
(InputStream input) or
(File input) or
(URL input) or
(String string)

md5() calculates MD5 value using the specified object and returns the result as a byte array.

sha(byte[] array {, int offset , int length } )or
(InputStream input) or
(File input) or
(URL input) or
(String string)

sha() calculates SHA value using the specified object and returns the result as a byte array.

e.g. 
md5("foo")   ==> [#9A, #87, #42, #07, #6E, #F9, #FF, #A5, #59, #1F, #63, #37, #04, #C2, #28, #6B]

sha("foo")   ==> [#C5, #F4, #E7, #20, #32, #50, #17, #36, #75, #41, #74, #35, #4B, #E3, #C6, #47, #D9, #71, #B8, #08]

hex(md5("foo")) ==> "9a8742076ef9ffa5591f633704c2286b"
hex(sha("foo")) ==> "c5f4e72032501736754174354be3c647d971b808"
