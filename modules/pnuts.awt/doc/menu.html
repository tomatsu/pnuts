Menu

menubar(Object[] list) or
(Frame frame {, String bundleName } , Object[] list) or
(JFrame jFrame {, String bundleName } , Object[] list)

menubar() creates a menu bar. If frame or jFrame is specified the menu bar is attached to the window. list should be an array which is structured as follows.

list ::= [menuList, ...]
menuList ::= [title, itemList, ...]
itemList ::= [string, function, shortcut] or [string, function]
shortcut ::= character or KeyStroke(only with Swing)

If bundleName is specified, labels are localized using formatMessage() function.

If jFrame is specified, it returns a javax.swing.JMenuBar object, otherwise, it returns a java.awt.MenuBar Object.

e.g. 
fr = frame("title")
menubar(fr, [["File",
              ["Open", open, 'o'],
              ["Save", save, 's'],
              ["Quit", quit, 'q']],
 	     ["Edit",
	      ["Cut", cut, 'x'],
	      ["Copy", copy, 'c'],
	      ["Paste", paste, 'v']]])
fr.show()
menu( { String bundleName, } Object[] menuList)

menu() creates a Menu object. If bundleName is specified, labels are localized using formatMessage() function.

e.g. 
menubar = ...
menubar.setHelpMenu(menu(["Help", ["About...", about]]))
fr = frame("title")
fr.setMenuBar(menuBar)
fr.show()
getMenuItem(JMenuBar menuBar , Object[] list) or
(MenuBar menuBar , Object[] list)

getMenuItem() refers to a menu item which is specified by a sequence of selections.

list ::= [string, ... ]
e.g. 
getMenuItem(menubar, ["File", "Open"]).setEnabled(false)
