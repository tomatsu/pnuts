Pnuts ユーザーズガイド -> pnuts.awt -> メニュー

メニュー

menubar(Object[] list) or
(Frame frame {, String bundleName } , Object[] list) or
(JFrame jFrame {, String bundleName } , Object[] list)

メニューバーを作成します。次のような配列の構造で階層的メニューを表します。

list ::= [menuList, ...]
menuList ::= [title, itemList, ...]
itemList ::= [string, function, shortcut] or [string, function]
shortcut ::= character or KeyStroke

bundleNameが指定された場合、formatMessage()関数を使ってラベルがローカライズされます。

frameOrJFrame が JFrame オブジェクトの場合、JMenuBar オブジェクトを返し、それ以外の場合、java.awt.MenuBar オブジェクトを返します。JMenuBar オブジェクトを返す場合、java.awt.swing.JMenuBar、com.sun.java.swing.JMenuBar の順にクラスを参照します。

例: 
fr = frame("title")
menubar(fr, [["File",
              ["Open", open, 'o'],
              ["Save", save, 's'],
              ["Quit", quit, 'q']],
 	     ["Edit",
	      ["Cut", cut, 'x'],
	      ["Copy", copy, 'c'],
	      ["Paste", paste, 'v']]])
fr.show()
menu( { String bundleName , } Object[] menuList)

メニューを作成します。 bundleNameが指定された場合、formatMessage()関数を使ってラベルがローカライズされます。

例: 
require("menu")
menubar = ...
menubar.setHelpMenu(menu(["Help", ["About...", about]]))
fr = frame("title")
fr.setMenuBar(menuBar)
fr.show()
getMenuItem(JMenuBar menuBar , Object[] list) or
(MenuBar menuBar , Object[] list)

メニューの選択項目を順に指定することで MenuItem または JMenuItemオブジェクトを求めます。

list ::= [string, ... ]

例: 
getMenuItem(menubar, ["File", "Open"]).setEnabled(false)
