キーマップ

defineKey(JComponent jComponent , String keystroke , PnutsFunction func { , int condition } ) or
(JComponent jComponent , String keystroke , Action action { , int condition } )

defineKey() は、キー入力keystrokeに対して、func またはactionが起動されるように登録します。 (Swing を利用することを前提にします。)

keystroke はKeyStroke::getKeyStroke(String)メソッドに渡されて、KeyStroke オブジェクトが作られます。

condition は次のいずれかです。

  • JComponent::UNDEFINED_CONDITION
  • JComponent::WHEN_ANCESTOR_OF_FOCUSED_COMPONENT
  • JComponent::WHEN_FOCUSED
  • JComponent::WHEN_IN_FUCUSED_WINDOW

例: 
defineKey(JTextPane(), "control alt K", function (e) println(e))
defineKey(JTextPane(), "released X", function (e) println(e))
getKeyStroke(String keystroke )

getKeyStroke() は、KeyStroke::getKeyStroke(String)メソッドを使って、文字列 keystrokeから swing の KeyStroke オブジェクトを作ります。
