スレッド・プール

threadPool ( int maxThreads { , int minThreads { long keepaliveInMSec } } )

threadPool()は、pnuts.multithread.ThreadPoolオブジェクトを返します。この返り値はasync()関数に渡され、非同期に実行されるスレッド間で、スレッドのプール(蓄積所)を共有するために使われます。

async(func(), threadpool)

async()は、指定されたスレッドプールにタスクを一つ追加します。そして、そのタスクの結果を取り出すことのできるハンドル・オブジェクトを返します。

スレッドプールにタスクのタスクは非同期に実行されます。タスクが終了するまで待って計算結果を取り出すには、ハンドル・オブジェクトを関数として呼び出します。

例: 
pool = threadPool(5)
handle = async(function () ..., pool)
handle()
