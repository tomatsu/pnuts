Thread Pool

threadPool ( int maxThreads { , int minThreads { long keepaliveInMsec } } )

threadPool() creates a org.pnuts.multithread.ThreadPool object, which will be passed to async() function so as to share a pool of threads by asynchronously executing tasks.

async(func(), threadpool)

async() adds a task to the specified thread pool and returns a handle object from which the result of the task can be obtained.

Tasks in a thread pool are executed asynchronously. To get the result of a asynchronous task, call the handle object as a function.

e.g. 
pool = threadPool(5)
handle = async(function () ..., pool)
