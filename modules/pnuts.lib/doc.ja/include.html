スクリプトの挿入

include(String scriptName)
includeFile(String fileName)

include() は、スクリプトscriptName を現在のコンテクストで実行します。 includeFile() スクリプトファイルを現在のコンテクストで実行します。 load() や loadFile()と違い、カレントパッケージやimport()状態はデフォルト値にリセットされません。

次の例では、["foo.bar", "java.lang.*", "*"] が表示されます。 

println(format(import()))
showImport.pnut
import("foo.bar")
function test(){
  includeFile("showImport.pnut")
}
test.pnut
