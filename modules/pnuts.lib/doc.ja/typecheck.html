型検査

isArray(Object object)
isFunction(Object object { , int narg } )
isGenerator(Object object )
iterable(Object object )

isArray() は、オブジェクトが配列かどうかを検査します。

例： 
isArray([1, 2]) ==> true
isArray(null) ==> false

isFunction() は、指定されたオブジェクトが関数かどうかを検査します。 narg が指定された場合は、その関数がnarg 個の引数を受け取れるかどうかを検査します。 可変長の引数は、-1で表されます。

例： 
isFunction(println, 0) ==> true
isFunction(println, -1) ==> true

isGenerator()は、指定されたオブジェクトがジェネレータかどうかを検査します。

例： 
isGenerator(range(1, 100)) ==> true

iterable() は、オブジェクトがfor文による繰り返し処理の対象となるかどうかを返します。

例： 
if (iterable(obj)){
  for (i : obj){
    handler(i)
  }
}
