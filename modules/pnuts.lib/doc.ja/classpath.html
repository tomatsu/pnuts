クラスパスの変更

setClassPath( Object[] directoryOrJarFile )

関数setClassPath()は、Pnutsがクラスを参照するときに使われるクラスパスを設定します。 関数 setClassPath() で設定されたディレクトリやJARファイルは、指定した順序で 環境変数 CLASSPATH よりも先に参照されます。 パラメータが配列の場合、配列の要素のディレクトリやJARファイルがクラスパスになります。

実行中に設定したクラスパスがリセットするには、パラメータにnull を指定します。

例： 
setClassPath(["k1.jar", "c:\\home"])
addClassPath( String path) or
( File path) or
( URL path)

addClassPath()は、現在のクラスパスに path を追加します。

例： 
addClassPath("foo.jar")
   => [file:/foo.jar]

addClassPath("bar.jar")
   => [file:/bar.jar, file:/foo.jar]
getClassPath()

getClassPath() 関数はクラスパスの設定を求めます。
