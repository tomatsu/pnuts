コンソール

コンソールへの出力

標準的なメッセージを表示するために、pnuts.libモジュールに以下の関数が定義されています。 関数の名前は java.io.PrintWriter クラスのメソッドと同じ名前です。

println(Object object, ...)
print(Object object, ...)
printAll(Enumeration enum) or
(Iterator iterator) or
(Generator generator) or
(Collection collection)
write(byte[] array , int offset , int length) or
(char[] array , int offset , int length)
write(byte[] array) or
(char[] array)
flush()

print() はobjectsを一つずつ表示します。 println() は、print()と同様ですが"line.separator" をストリームに追記します。 write()は、ストリームクラスの writeメソッドを呼び出します。

flush()は、バッファの内容をフラッシュします。

printAll()はパラメータの要素をすべて表示します。

getWriter()

getWriter()は、現在のコンソールメッセージ出力用のPrintWriterオブジェクトを返します。

エラーの表示

error(String string {, Context context })

error()は、エラーメッセージを表示します。contextが指定された場合は、そのコンテクストのエラー出力にメッセージが表示されます。

getErrorWriter()

getErrorWriter()は、現在のエラーメッセージ出力用のPrintWriterオブジェクトを返します。

setVerbose( boolean verbose )

setVerbose() は、冗長モードを有効化または無効化します。
