日付／時刻

currentTimeMillis()

currentTimeMillis() は、現在の時刻をミリ秒(long)で返します。

date() or
(long timeMillis) or
(Date date)or
(String dateString { , String pattern }) or
(int year, int month, int day { , int hour, int minute, int second } )

date() は、java.util.Date オブジェクトを返します。引数がない場合、現在時刻を表すオブジェクトを返します。

dateString が指定された場合、SimpleDateFormat に説明されている規則に従って解析されます。pattern のデフォルト値は"MM/dd/yyyy"です。

例： 
date("2000/10/27", "yyyy/MM/dd")
