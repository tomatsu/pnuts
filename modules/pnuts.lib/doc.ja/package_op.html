パッケージ

findPackage(String packageName)

findPackage() 関数は、指定した名前の Package オブジェクトを求め ます。 指定したパッケージがない場合は null を返します。 

findPackage("foo")   ==> null
package("foo")       ==> package "foo"
package("")
findPackage("foo")   ==> package "foo"
package()            ==> package ""
getPackage(String packageName)

getPackage() は、指定した名前のPackage オブジェクトを求め ます。指定したパッケージがない場合は、そのパッケージを作成してそれを返します。 

package()             ==> package ""
findPackage("bar")    ==> null
getPackage("bar")     ==> package "bar"
package()             ==> package ""
createPackage()
createPackage( ` expression ; ... `)
createPackage( expression , ... )

createPackage()は、匿名のパッケージを作成して返します。

引数の数 型 動作
1 java.lang.String パッケージ内で文字列を評価する
1 java.lang.Object[] 各要素を[key, value] のペアと解釈して、パッケージでそれらを定義する。
任意 任意 引数の中に関数がある場合は、パッケージでその関数を関数名で登録する
例: 
p = createPackage()
p.x = 1
p.y = 2

q = createPackage("x = 1; y = 2")
q.x     ==> 1
q.y     ==> 2

r = createPackage([["a", "hello"], ["b", "world"]])
r.a  ==> "hello"
r.b  ==> "world"

createPackage()は、通常の関数とちがって、呼び出し時のコンテクストで引数を評価します。「スクリプトパッケージ」 を参照。

removePackage(String name)

removePackage() は指定したパッケージを除外します。

rootScope( )

rootScope()は、パッケージ階層の最上位のパッケージを返します。 通常はグローバル・パッケージです。
