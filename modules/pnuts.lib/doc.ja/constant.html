定数の定義

constant . name = value
constant [ name ] = value

この式は、カレントパッケージに定数nameを定義します。一度その変数が定義されると、値を修正することはできません。 

constant.PI = 3.14
PI         ==>  3.14
PI = 3     ==>  error
constant( { Package pkg } )

この関数は、指定されたパッケージpkgに定数を定義するためのハンドルオブジェクトをかえします。pkgが指定されない場合はカレントパッケージが使われます。一度その変数が定義されると、値を修正することはできません。 

pkg = $()
const = constant(pkg)
const.PI = 3.14
pkg.PI       ==> 3.14
pkg.PI = 3   ==> error
