タイマー

schedule(task() , delayOrTime { , period } )

タイマーを使って関数task() を後で実行します。 java.util.Timerオブジェクトを返します。Timer.cancel()メソッドを呼び出すことでタイマーをキャンセルできます。

delayOrTimeには、遅延（ミリ秒）または、実行する時刻を表すDateを指定します。periodを指定した場合、periodミリ秒ごとにtask()が実行されます。 

t = schedule(function (){ println(date()); flush()}, 0, 1000)
...
t.cancel()
