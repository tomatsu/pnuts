VMのシャットダウン

exit( { exitValue })

exit()は、VMのOSプロセスを停止させます。exitValueに整数を指定した場合は、その値がプロセスの終了コードになります。

シャットダウン・フック

これらの関数は、JDK1.3 以降で利用できます。

addShutdownHook(hookFunction () )

addShutdownHook() は、指定された関数をシャットダウン・フックとして登録し、その関数に対応するThread オブジェクトを返します。

removeShutdownHook(Thread hookThread )

removeShutdownHook() は、登録されているシャットダウン・フックを解除します。
