Shutdown Hook

Terminating JVM

exit( { exitValue })

exit() terminates an OS-process for the JVM. When an integer is specified for exitValue, it becomes the exit code of the terminated process.

Shutdown Hook

These functions are available on JDK1.3 or higher.

addShutdownHook(hookFunction () )

addShutdownHook() register the specified function as a shutdown hook and returns a Thread object associated with the hook function.

removeShutdownHook(Thread hookThread )

removeShutdownHook() unregisters a previously-registered shutdown hook.
