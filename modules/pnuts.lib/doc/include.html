Script Inclusion

include(String scriptName)
includeFile(String fileName)

include() executes a script named scriptName in the current context. includeFile() execute a script file in the current context. Unlike load() and loadFile(), current package and import() state are not reset to the default setting.

In the following example, ["foo.bar", "java.lang.*", "*"] is printed. 

println(format(import()))

import("foo.bar")
function test(){
  includeFile("showImport.pnut")
}
