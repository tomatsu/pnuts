String Representation of Objects

hex(Object integerOrByteArray)
string(Object object)
format(Object object)

hex() returns a hexiadecimal format string constructed from the specified integer or byte array.

string() returns a String representation of the specified object. An empty string is returned if object is null.

format() returns a formatted string constructed from the specified object.

e.g. 
string([1,2,3]) ==> "[Ljava.lang.Object;@17a37c"
format([1,2,3]) ==> "[1, 2, 3]"
hex(10) ==> "a"
hex((byte[])[#ca, #fe, #ba, #be]) ==> "cafebabe"
