Converting Character Case

toLowerCase((char | String | CharSequence) charData)
toUpperCase((char | String | CharSequence) charData)

toLowerCase() converts character data in the given object to lower case. toUpperCase() converts character data in the given object to upper case. 

toLowerCase("ABC") ==> "abc"
