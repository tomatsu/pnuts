Constants

constant . name = value
constant [ name ] = value

This expression defines a constant variable named name in the current package. Once the variable is defined, it can not be modified. 

constant.PI = 3.14
PI         ==>  3.14
PI = 3     ==>  error
constant( { Package pkg } )

This function returns a handle object that is used to define constants in the specified package, or the current package if pkg is not specified. Once the variable is defined, it can not be modified. 

pkg = $()
const = constant(pkg)
const.PI = 3.14
pkg.PI       ==> 3.14
pkg.PI = 3   ==> error
