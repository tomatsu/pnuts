Date/Time

currentTimeMillis()

currentTimeMillis() returns the current time in millisenconds.

date() or
(long timeMillis) or
(Date date) or
(String dateString { , String pattern }) or
(int year, int month, int day { , int hour, int minute, int second } )

date() returns a java.util.Date object. If no argument is given, it returns a object that represents the current time.

When dateString is specified, it is parsed under the rule described at SimpleDateFormat. The default value of pattern is "MM/dd/yyyy".

e.g. 
date("2000/10/27", "yyyy/MM/dd")
