Timer

schedule(task() , delayOrTime { , period } )

Schedules the specified function-call task() and returns a java.util.Timer object.

Timer.cancel() discards the scheduled task.

delayOrTime should be a long value that represents 'delay' or a Date object that indicates when the task will be executed. If period is specified, task() is repeatedly executed for every period milliseconds. 

t = schedule(function (){ println(date()); flush()}, 0, 1000)
...
t.cancel()
