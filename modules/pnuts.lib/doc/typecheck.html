Type Check

isArray(Object object)
isFunction(Object object { , int narg } )
isGenerator(Object object )
iterable(Object object )

isArray() checks if the object is an array.

e.g. 
isArray([1, 2]) ==> true
isArray(null) ==> false

isFunction() checks if the object is a function. When narg is specified checks if the function can be called with narg parameters. -1 is used for arbitrary number of parameters.

e.g. 
isFunction(println, 0) ==> true
isFunction(println, -1) ==> true

isGenerator() checks if the object is a generator.

e.g. 
isGenerator(range(1, 100))   ==> true

iterable() checks if the object can be used in 'for' statement.

e.g. 
if (iterable(obj)){
   for (i : obj){
     handler(i)
   }
}
