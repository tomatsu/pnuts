File

getFile(String urlString { , String path } ) or
(File file { , String path })

getFile() returns a java.io.File object. If the 2nd argument path is specified, it returns a relative File from the one created from the 1st argument.

URL

getURL(String urlString { , String path } ) or
(File file { , String path }) or
(URL url { , String path })

getURL() returns a URL object converted from the argument. When path is specified, it returns a relative URL from the one created from the 1st argument.

Resource

See also http://java.sun.com/j2se/1.4/docs/guide/resources/ for information on resources.

getResource(String name)

getResource() returns a URL Object that represents the specified resource.

e.g. 
rsrc = getResource("pnuts/lib/init.pnut")
read(rsrc.openStream())

printAll(getResource("pnuts/lib/init.pnut"))

File Name

basename( (String | File　) filename { , String suffix } )

Given a string containing a path to a file, this function will return the base name of the file. If the filename ends in suffix this will also be cut off. 

basename("/htdocs/index.html") ---> "index"
basename("/htdocs/index.html", ".html") ---> "index"
basename("/htdocs/index.html", ".txt") ---> "index.html"
dirname( (String | File　) path )

Given a string containing a path to a file, this function will return the name of the directory. 

dirname("/etc/passwd") ---> /etc
dirname("/usr/local/bin/") ---> /usr/local
