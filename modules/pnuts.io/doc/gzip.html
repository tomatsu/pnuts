Compression/Decompression with GZIP

zcat(input {, output })

zcat() decompress the content of input. If output is specified, data is written to the object. Otherwise, it returns a java.io.PipedInputStream object which can be read with read() function.

input must be either File, String(file name), InputStream, or URL. output must be either File, String(file name), or OutputStream.

The following example illustrates how to load a gzip-compressed script. 

load(zcat("big.pnut.gz"))
gzip(input {, output })

gzip() compress the content of the file input. If output is specified, data is written to the object. Otherwise, it returns a java.io.PipedInputStream object which can be read with read() function.

input must be either File, String(file name), InputStream, or URL. output must be either File, String(file name), or OutputStream.
