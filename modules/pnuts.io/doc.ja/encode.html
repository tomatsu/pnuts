エンコーダ・デコーダ

base64encode(InputStream input {, OutputStream output })
base64decode(InputStream input {, OutputStream output })
base64encode(byte[] input)
base64decode(String input)

base64encode()は、InputStreamが指定された場合、inputからデータを読み込んでBASE64でエンコードします。 outputが指定された場合、エンコードされたデータがそこに書き込まれます。 指定されない場合、java.io.PipedInputStream オブジェクトを返します。後でread() 関数で読み出すことができます。

パラメータにbyte配列が指定された場合は、BASE64でエンコードした結果の文字列を返します。

base64decode()は、 InputStreamが指定された場合、inputからデータを読み込んでBASE64でデコードします。 outputが指定された場合、デコードされたデータがそこに書き込まれます。 指定されない場合、java.io.PipedInputStream オブジェクトを返します。後でread() 関数で読み出すことができます。

パラメータに文字列が指定された場合は、BASE64でデコードした結果のbyte配列を返します。 

read(base64encode(open("pnuts.bat")))

これらの関数は、それぞれ sun.misc.BASE64Encoder クラスと sun.misc.BASE64Decoder クラスに依存します。

uuencode(InputStream input {, OutputStream output {, String name } })
uudecode(InputStream input {, OutputStream output })

uuencode()は、 inputストリームからデータを読み込んでuuencodeでエンコードします。 outputが指定された場合、エンコードされたデータがそこに書き込まれます。 指定されない場合、java.io.PipedInputStream オブジェクトを返します。後でread() 関数で読み出すことができます。 nameが指定された場合、その名前が出力に反映されます。

uudecode()は、 inputストリームからデータを読み込んでuudecodeでデコードします。 outputが指定された場合、デコードされたデータがそこに書き込まれます。 指定されない場合、java.io.PipedInputStream オブジェクトを返します。後でread() 関数で読み出すことができます。

これらの関数は、それぞれ sun.misc.UUEncoder クラスと sun.misc.UUDecoder クラスに依存します。

stringToUnicode(String string)
unicodeToString(String string)

stringToUnicode() は、文字列を unicode-escape されたシーケンスに変換されます。

unicodeToString() は unicode-escape されたシーケンスを対応する文字列に変換します。
