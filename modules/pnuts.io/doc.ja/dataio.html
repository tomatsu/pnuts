データI/O

dataInput(InputStream in)
dataOutput(OutputStream out)

dataInput() はDataInputStream オブジェクトを返します。 dataOutput() はDataOutputStream オブジェクトを返します。

writeBoolean(DataOutputStream out, boolean trueOrFalse)
writeShort(DataOutputStream out, int s)
writeChar(DataOutputStream out, int ch)
writeChars(DataOutputStream out, String str)
writeInt(DataOutputStream out, int n)
writeLong(DataOutputStream out, long n)
writeFloat(DataOutputStream out, float n)
writeDouble(DataOutputStream out, double n)
writeUTF(DataOutputStream out, String str)

指定されたDataOutputStream へ、それぞれに対応する型の値を一つ書き込みます。

例： 
dout = dataOutput(open("test.dat", "w"))
dout.writeInt(100)
dout.close()
readBoolean(DataInputStream in)
readUnsignedByte(DataInputStream in)
readShort(DataInputStream in)
readUnsignedShort(DataInputStream in)
readChar(DataInputStream in)
readInt(DataInputStream in)
readLong(DataInputStream in)
readFloat(DataInputStream in)
readDouble(DataInputStream in)
readUTF(DataInputStream in)

指定されたDataInputStream から、それぞれに対応する型の値を一つ読み込みます。

例： 
din = dataInput(open("test.dat"))
println(din.readInt())
din.close()
