シリアライゼイション

writeObject(Serializable object , String filename) or
(Serializable object , File file) or
(Serializable object , OutputStream output)

writeObject() は object をシリアライズします。 fileかfilenameが指定された場合はそのファイルに保存します。 outputが指定された場合はそのOutputStream に書き込みます。

readObject(String fileName) or
(File file) or
(InputStream input)

fileまたはfilename が指定された場合、そのファイルからシリアライズされたオブジェクトを読み込みます。 inputが指定された場合は、そのInputStreamからシリアライズされたオブジェクトを読み込みます。
