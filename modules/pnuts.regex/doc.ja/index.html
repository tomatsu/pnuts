pnuts.regexモジュール

pnuts.regex モジュールは、 特定の正規表現APIに依存しない形で、正規表現の関数を提供します。

J2SE1.4では、デフォルトで標準の正規表現APIが使われます。 それ以前のJREではサードパーティが提供する正規表現パッケージが必要になります。

システム・プロパティpnuts.regex.moduleが定義されている場合はそのモジュールを使用し、定義されていない場合はプロパティファイル"pnuts/regex/regex.properties"にあらかじめ指定されたリストの中から利用可能な正規表現APIを探します。

正規表現API モジュール名
JSR51 (J2SE1.4.X) pnuts.regex.jsr51

Jakarta ORO jakarta-oro
Regex4J regex4j
Jakarta Regexp jakarta-regexp
GNU Regexp gnu-regexp
Regexp API in Xerces XML parser xerces-regexp
