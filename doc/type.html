Primitive Type Conversion

byte(Number number) or (String string) or (char character)
int(Number number) or (String string) or (char character)
short(Number number) or (String string)
long(Number number) or (String string)
float(Number number) or (String string)
double(Number number) or (String string)
char(Number number)

Converts a given object to an object that represents the corresponding primitive data. If the conversion fails, returns null.

e.g. 
byte(-126)    ==> new Byte(0x82)
int("123") ==> new Integer(123)
int("hello") ==> null
double("123.456") ==> new Double(123.456)
char(65)   ==> 'A'
