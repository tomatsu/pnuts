JSP タグライブラリ

Pnutsスクリプトを実行するためのJava Server Pagesのタグライブラリがpnuts-taglib.jarにふくまれています。 

<%@ taglib uri="/pnuts-taglib.jar" prefix="pnuts" %>

<pnuts:eval>
  expression
</pnuts:eval>

<pnuts:eval> タグは expression を評価して、そのエレメントが結果で置き換えられます。

例 (${TOMCAT}/webapps/test/test.jsp) 
<%@ taglib uri="/pnuts-taglib.jar" prefix="pnuts" %>
<HTML>
<HEAD>
<TITLE></TITLE> 
</HEAD>
<BODY BGCOLOR="#FFFFFF"> 

<pnuts:eval>
  use("pnuts.util")
  foo = 100
</pnuts:eval>

<pnuts:eval>
  println(foo)
</pnuts:eval>
</BODY> 
</HTML>
