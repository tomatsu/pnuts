ユーザーズガイドの書式

ユーザーズガイドでは、関数の使い方を説明するときに次のような書式にしたがいます。

loadFile( String fileName )

この例では、loadFile という名前の関数は String 型の変数を一つ受け取ることを示しています。

シグネチャ（パラメータの型の並び）が複数ある関数については、引数部分を ' or 'をはさんで記述します。

write(byte[] array , int offset , int length) or
(char[] array , int offset , int length)
' or ' で引数部分を連結する例

オプションとなる引数がある場合は、その引数を括弧 {} で囲んで記述します。

open(String fileName {, String mode } )
オプション引数の例

パラメータがObject型や配列の場合は、要素の型を省略することがあります。

sort(array [ ] )

パラメータが関数の場合、パラメータリストを括弧で囲んだものが後ろにきます。

memcache(handler ( key ) )
