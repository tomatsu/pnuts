パッケージ

パッケージとは、変数や関数を定義する独立した名前空間です。

package( ) or
(String packageName) or
(Package aPackage)

1引数のpackage() 関数はカレントパッケージを変更します。 0引数のpackage() 関数はカレントパッケージに対応するPackage オブジェクトを返します。 

package("foo")   ==> package "foo"
package()        ==> package "foo"
package("")      ==> package ""
package()        ==> package ""
use(String packageName)

pkgNameで示されるパッケージで定義されているすべての関数をカレントパッケージで直接参照できるようにします。モジュールは、この組み込み関数を利用して実現されています。 

package("foo")
function x() 100
y = 200

package("bar")
use("foo")
x ==> function x
y ==> エラー

package("")
x ==> エラー
y ==> エラー
aPackage . symbol
aPackage . symbol = value
foo = 100
package().foo     ==> 100
package().foo = 200
foo               ==> 200
package().bar     ==> null
aPackage .defined (String symbolName, Context context )

Package.defined メソッドは、指定した名前のシンボルがターゲットのパッケージで定義されているかどうかを調べます。 

getPackage("foo")
context = getContext()
findPackage("foo").defined("bar", context)   ==> false
foo::bar = 100
findPackage("foo").defined("bar", context)   ==> true
aPackage .clear (String symbolName, Context context )

Package.clear メソッドは指定したシンボルをターゲットのパッケージから削除します。 

context = getContext()
findPackage("foo").defined("bar", context)   ==> true
findPackage("foo").clear("bar", context)
findPackage("foo").defined("bar", context)   ==> false
