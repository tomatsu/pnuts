TomcatとApacheの連携

設定例(Tomcat4.0.XとApache1.3.Xとmod_jk)

/usr/local/apache/conf/httpd.conf 
...
Include /usr/local/jakarta-tomcat-4.0.4/conf/mod_jk.conf

/usr/local/jakarta-tomcat-4.0.4/conf/mod_jk.conf 
LoadModule jk_module libexec/mod_jk.so

<IfModule mod_jk.c>
JkWorkersFile /usr/local/jakarta-tomcat-4.0.4/conf/workers.properties
JkLogFile  logs/jk.log
JkLogLevel error
JkMount /*.pnut ajp13
JkMount /WEB-INF/* ajp13
</IfModule>

/usr/local/jakarta-tomcat-4.0.4/conf/workers.properties 
workers.tomcat_home=/usr/local/jakarta-tomcat
workers.java_home=/usr/java/j2sdk1.4.0
ps=/
worker.list=ajp12, ajp13

worker.ajp13.port=8009
worker.ajp13.host=localhost
worker.ajp13.type=ajp13


% cd /usr/local/jakarta-tomcat-4.0.4
% mv webapps webapps.bak
% mkdir webapps
% cd webapps
% ln -s /usr/local/apache/htdocs ROOT

参考： http://www.ingrid.org/jajakarta/tomcat/tomcat-3.2.3/doc-ja/tomcat-apache-howto.html
