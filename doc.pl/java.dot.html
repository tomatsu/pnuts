Moduły java.* 

Kiedy zostaję użyta komenda  pnuts wykonująca skrypt domyślnie moduł  pnuts.tools jest ładowany, który wczytuje następujące moduły.

Nazwa modułu Klasy
java.net URL, URI, Socket, ServerSocket, MulticastSocket, Authenticator, DatagramSocket, DatagramPacket
java.util Date, BitSet, Calendar, Hashtable, Vector, Stack, Map, List, Set, Collection, Collections, Arrays, Random, Locale, TimeZone, Enumeration, Iterator
java.io File, OutputStream, InputStream, IOException, Reader, Writer

Od czasu gdy eksportowane nazwy z modułów mają większy priorytet niż importowane klasy funkcja import nie może nadpisać eksportowanej nazwy z modułu. Przykładowo odwołanie do klasy  java.awt.List  zamiast do klasy  java.util.List wymaga wyrejestrowania modułu  java.util z listy modułów a następnie zaimportowania klasy java.awt.List. 

unuse("java.util")
import java.awt.List
