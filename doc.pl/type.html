Podręcznik Pnuts -> Konwersje typów prostych

Konwersje typów prostych

byte(Number liczba) lub (String napis) or (char znak)
int(Number liczba) lub(String napis) or (char znak)
short(Number liczba) lub (String napis)
long(Number liczba) lub (String napis)
float(Number liczba) lub (String napis)
double(Number liczba) lub (String napis)
char(Number liczba)

Konwertuje podany obiekt do odpowiedniego typu prostego, jeżeli konwersja nie jest możliwa zwracane jest null.

np. 
byte(-126)    ==> new Byte(0x82)
int("123") ==> new Integer(123)
int("hello") ==> null
double("123.456") ==> new Double(123.456)
char(65)   ==> 'A'
