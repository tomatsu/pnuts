Dostosowywanie zachowania wyrażeń podstawowych

Zachowanie pól klas, dostępu indeksowego, wywołania  metod/construktorów oraz operatorów może być zmienione przez atrybuty obietku  Context.

Dostęp do pól
target . field
target :: field
Dostęp indeksowy target [ index ]
Wywołanie metod
target . method (...)
target :: method (...)
Konstruktor
target (...)
Operatory +,-,*,/,&,|,^,<<,>>,>>>,&&,||,++,--

Ustawanie konfiguracji

Konfiguracja kontekstu może być ustawiona przez metodę  Context.setConfiguration(Configuration). 

Context context = new Context();
context.setConfiguration(new NonPublicMemberAccessor());
...

Klasa pnuts.lang.Configuration 

pnuts.lang.Configuration definiuje domyślne zachowanie wyrażeń. W podklasie można zdefiniować własne zachowanie przez nadpisanie odpowiednich metod.

Dostęp do pól

public Object getField( Context context , Object target , String name );
public void putField( Context context, Object target , String name, Object value );
public Object getStaticField( Context context, Class target , String name );
public void putStaticField( Context context, Class target , String name, Object value );

Dostęp indeksowy

public Object getElement( Context context, Object target , Object key );
public void setElement( Context context , Object target , Object key, Object value);

Konstruktor

public Object callConstructor( Context context, Class c , Object[] args , Class[] types );

Wywołanie metod

public Object callMethod( Context context , Object target , Object[] args , Class[] types );

Operatory

Działanie operatorów może być ustawione przez ustawianie zabezpieczonych zmiennych w konstruktorze.
protected BinaryOperator _add, _subtract, _multiply, _divide, _mod, _and, _or, _xor, _shiftLeft, _shiftRight, _shiftArithmetic;
protected UnaryOperator _add1, _subtract1, _negate, _not;
protected BooleanOperator _lt, _le, _gt, _ge, _eq;

Predefiniowana konfiguracja

Poniższe klasy zdefiniowane w pakiecie pnuts.lang package ustalają domyślną konfigurację.

pnuts.lang.Configuration
Domyślna konfiguracja z dostępem do własciwości i metod Beans,zmiennych kontekstu oraz atrybutów  pnuts.lang.Property.  Pod platforma Java 2 dostęp do wpisów w obiektach Map działa jak dostęp do pół obiektu.
pnuts.ext.PublicMemberAccessor
Tak samo jak domyślna z wyjatkiem że jest dostęp do publicznych pól zamiast właściwości Beans.
pnuts.ext.NonPublicMemberAccessor
Pozwala na dotęp do niepublicznych motod i pól, wymaga platformy Java 2.
pnuts.awt.EventQueueConfiguration
Używan wątku  zdarzeń AWT w dostępnie do API.

Właściwość pnuts.lang.defaultConfiguration 

Jeżeli systemowa właściwość pnuts.lang.defaultConfiguraion wskazuje na prawidłową podlasę konfiguracji jest ona użyta jako domyślna konfiguracja.

Przykładowo jezeli uruchomimy interpreter jak podano poniżej wszytskie pola w klasach będą dostępne jako publiczne. 

% java -Dpnuts.lang.defaultConfiguration=pnuts.lang.PublicMemberConfiguration pnuts.tools.Main
Zobacz również 'Configuracja kontekstu'.

Funkcje i konfiguracja

Funkcje zdefiniowane w skryptach są wykonywane w konfiguracji w której zostały zdefiniowane  (konfiguracje w której skrypt został zaladowany)
