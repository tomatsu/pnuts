Podręcznik Pnuts -> Pakiety

Pakiety

Pakiet jest przestrzenią nazw identyfikowaną po nazwie gdzie zdefiniowane są funkcje i zmienne.

package( ) lub
(String nazwaPakietu) lub
(Package pakiet)

package() wywołane z jednym argumentem zmienia bieżący pakiet na wskazany w parametrze. package()wywołane bez parametru zwraca bieżący pakiet jako obiekt  Package . 

package("foo")   ==> package "foo"
package()        ==> package "foo"
package("")      ==> package ""
package()        ==> package ""
use(String nazwaPakietu)

Ta wbudowana funkcja importuje wszystkie funkcje z  modułu, nazwaPakietu. Zaimportowane funkcje mogą być wołane bez podawania nazwy pakietu. 

package("foo")
function x() 100
y = 200

package("bar")
use("foo")
x ==> function x
y ==> error

package("")
x ==> error
y ==> error
pakiet . symbol
pakiet . symbol = wartość
foo = 100
package().foo     ==> 100
package().foo = 200
foo               ==> 200
package().bar     ==> null
pakiet .defined (String nazwaSymbolu , Context kontekst )

Package.defined sprawdza czy podany symbol jest zdefiniowany w danym pakiecie. 

getPackage("foo")
context = getContext()
findPackage("foo").defined("bar", context)   ==> false
foo::bar = 100
findPackage("foo").defined("bar", context)   ==> true
pakiet .clear (String nazwaSymbolu, Context kontekst )

Package.clear metoda kasuje podany symbol z pakietu. 

context = getContext()
findPackage("foo").defined("bar", context)   ==> true
findPackage("foo").clear("bar", context)
findPackage("foo").defined("bar", context)   ==> false
